# [栈内存与堆内存的区别](https://www.cnblogs.com/leitongwangtingting/articles/14652144.html)

栈 (Stack): 由编译器自动分配和释放，存放函数的参数，局部变量、临时变量、函数返回地址等。

堆 (Heap): 由程序员分配和释放，对于 C/C++ 必须要手动释放开辟的堆空间，否则会引起内存泄漏。
