# 方案一: 贪心

![p9bc43t.png](https://s1.ax1x.com/2023/05/26/p9bc43t.png)

```js
public boolean canPlaceFlowers(int[] flowerbed, int n) {
	for (int i = 0, len = flowerbed.length; i < len && n > 0;) {
		if (flowerbed[i] == 1) {
			i += 2;
		} else if (i == flowerbed.length - 1 || flowerbed[i + 1] == 0) {
			n--;
			i += 2;
		} else {
			i += 3;
		}
	}
	return n <= 0;
}
```
