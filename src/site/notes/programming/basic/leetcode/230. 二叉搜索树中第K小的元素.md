# Solution Tips

# 方案一: 中序迭代

迭代法可以提前退出, 不用遍历全部节点

```js
var kthSmallest = function(root, k) {
    // 迭代中序试试

    const queue = [root];
    let cur = root;
    let index = 0;

    while (cur || queue.length) {
        while (cur) {
            queue.push(cur);
            cur = cur.left;
        }

        cur = queue.pop();
        index++;
        if (index === k) return cur.val;
        cur = cur.right;
    }

    return;
};
```
