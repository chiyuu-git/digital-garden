## 点击其他区域, 防止 Input 失焦

在开发过程中，总会碰到以下两个情况：

1. 要求点击某个区域，阻止 input 框 (或者设置了 contenteditable=“true” 的编辑区) 失去焦点。
2. 要求像微信输入框那种点击选择表情、图片等要求保留原来焦点的位置的情况。因为一失去焦点位置，除非是往最前面或最后面插入，否则要想在原来的地方插入内容是很难的。
